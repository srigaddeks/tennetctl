# 01_reports — Scope

The Custom Reports sub-feature allows users to create branded, modular reports for clients or internal stakeholders.

## In Scope
- **Modular Report Builder**: Adding charts, tables, and metric summaries as "blocks".
- **Custom Branding**: Uploading a logo and setting the primary report color.
- **Exporting**: Generating high-quality PDF or PNG versions of the report.
- **Recurring Reports**: Scheduling reports to be generated and emailed automatically.

## Acceptance Criteria
- [ ] Reports must allow for a custom "Description" or "Notes" field for each block.
- [ ] Must support date-comparison views (e.g., this period vs. last period).
