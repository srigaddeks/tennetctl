# 01_mobile_extensions — Scope

The Mobile Companion (iOS/Android) ensures continuous content creation and management, specifically for platforms that restrict full scheduling automation.

## In Scope
- **Share to Buffer**: A mobile system extension allowing users to "send" a URL or photo from any app to the Buffer drafts.
- **Push Reminders**: Mobile notifications for non-automated content (Instagram Reels, TikToks).
- **Draft & Idea Capturing**: Quick-edit of text/images for mobile-first content (e.g., street photography, event coverage).
- **Biometric Security**: FaceID/TouchID unlocking for the mobile application.

## Out of Scope
- **Complex Analytics Dashboarding**: (Mobile only) basic metric overview is included, but full report building is a Desktop feature.
- **Organization Settings**: (Mobile only) manage members or billing is not available on mobile.

## Acceptance Criteria
- [ ] Push notifications must include the original media (video/photo).
- [ ] Users must be able to "Copy Caption" with one tap during a reminder.
- [ ] Share extension must support multiple image uploads from the gallery.
