# 01_analytics — Scope (Start Page)

This sub-feature tracks the performance of the Start Page microsite, focusing on traffic sources and link conversion.

## In Scope
- **Traffic Overview**: Total Views, Unique Visitors, and Click-Through Rate (CTR).
- **Link Performance**: A ranked list of the most-clicked links on the page.
- **Referral Sources**: Identifying where visitors are coming from (e.g., Instagram Bio, LinkedIn Profile).

## Acceptance Criteria
- [ ] Must distinguish between "Total Clicks" and "Unique Clicks" per link.
