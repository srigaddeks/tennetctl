# 01_editor — Scope

The Start Page Editor allows users to build a high-converting, mobile-optimized microsite (link-in-bio) for their social profiles.

## In Scope
- **Block-Based Builder**: Adding, removing, and reordering blocks (Links, Text, Media, Social Icons).
- **Styling & Customization**: Choosing themes, custom font pairings, and brand color presets.
- **Button Animations**: Subtle micro-animations to drive click-through rates.
- **E-commerce Widgets**: (e.g., Shop blocks or featured product cards).
- **Public URL Management**: Mapping a custom subdomain (buffer.com/sri) or domain.

## Out of Scope
- **Full Website Hosting**: It is a landing page tool, not a full CMS (like WordPress).
- **Email Collection**: (Varies by tier) high-level marketing automation is out of scope.

## Acceptance Criteria
- [ ] Editor must provide a mobile-responsive preview that updates in real-time.
- [ ] Must validate URL formats in the "Link" blocks.
- [ ] Changes must be "Published" manually to go live (Draft state support).
