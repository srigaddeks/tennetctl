# 01_ideas — Scope

The Ideas Board is a Kanban-style workspace for brainstorming content before it enters the scheduling pipeline.

## In Scope
- **Idea Cards**: Lightweight cards for text, image links, or rough concepts.
- **Kanban Board**: Drag-and-drop columns (e.g., "To do", "Drafting", "Ready").
- **AI Idea Generator**: A prompt-based tool to generate 10+ content concepts.
- **Convert to Draft**: One-click action to move an Idea Card into the Post Composer.

## Acceptance Criteria
- [ ] Ideas must remain separate from the queue until manually converted.
- [ ] Must support labels/tags for categorizing ideas.
