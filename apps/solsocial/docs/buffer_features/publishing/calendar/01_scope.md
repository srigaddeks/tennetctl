# 01_calendar — Scope

The Calendar View provides a chronological visual interface for content management, allowing users to spot gaps in their posting schedule.

## In Scope
- **Visual Grid**: Weekly and Monthly views of scheduled posts across all profiles.
- **Drag-and-Drop Rescheduling**: Moving posts between dates and times via the calendar interface.
- **Slot Creation**: Quick-add buttons on empty date/time cells to launch the Post Composer pre-filled for that slot.
- **Filters**: Toggling visibility based on Channel, Campaign, or Post State (Draft vs. Scheduled).

## Out of Scope
- **Calendar External Sync**: (e.g., Google Calendar sync) is not part of the core product.

## Acceptance Criteria
- [ ] Calendar must update in real-time when a post is scheduled via the Composer.
- [ ] Must distinguish between network types using icons on each calendar card.
- [ ] Navigation must support "Jump to Today" and date picking.
