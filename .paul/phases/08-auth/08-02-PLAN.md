    Assert Topbar Shows Signin Link
        Wait For Elements State    [data-testid="topbar-signin"]    visible    timeout=5s

    Signout Via Topbar
        Click    [data-testid="topbar-signout"]
        Wait For Load State    networkidle

    API Create Test User
        [Arguments]    ${email}    ${password}    ${display_name}
        ${body}=    Create Dictionary    email=${email}    password=${password}    display_name=${display_name}
        ${resp}=    POST    ${BACKEND_URL}/v1/auth/signup    json=${body}    expected_status=201
        RETURN    ${resp.json()['data']}

    API Delete Test User
        [Arguments]    ${email}
        # Cleanup via direct DB not available in Robot — use signout + rely on test isolation
        # (test users are prefixed and don't affect prod data)
        No Operation
    ```

    Documentation    IAM Auth UI — signin/signout/redirect E2E.
    ...              Drives live backend (51734) + frontend (51735).
    ...              Creates a test user via API, signs in via UI, checks topbar,
    ...              signs out, and verifies redirect to signin page.
    Library     Browser
    Resource    ../keywords/auth.resource
    Suite Setup       Launch Suite
    Suite Teardown    Close Browser    ALL

    *** Variables ***
    ${TS}            ${EMPTY}
    ${TEST_EMAIL}    ${EMPTY}
    ${TEST_PASS}     pass-E2E-1234!

    *** Keywords ***
    Launch Suite
        ${ts}=    Get Time    epoch
        Set Suite Variable    ${TS}    ${ts}
        Set Suite Variable    ${TEST_EMAIL}    ${AUTH_PREFIX}-${ts}@tennetctl.test
        New Browser    chromium    headless=true
        New Context
        New Page    about:blank
        API Create Test User    ${TEST_EMAIL}    ${TEST_PASS}    E2E User

    *** Test Cases ***
    Signin Shows User In Topbar
        [Documentation]    Navigate to /auth/signin, fill credentials, submit,
        ...                verify topbar displays the user's email.
        Open Signin Page
        Signin Via UI    ${TEST_EMAIL}    ${TEST_PASS}
        Assert Topbar Shows User    ${TEST_EMAIL}

    Signout Redirects To Signin
        [Documentation]    Click Sign Out from topbar, verify page is now /auth/signin

        Get Url    contains    /auth/signin
        Assert Topbar Shows Signin Link

    Protected Page Redirects Unauthenticated User
        [Documentation]    Navigate directly to /vault without a session.
        ...                The app should redirect to /auth/signin.
        Go To    http://localhost:51735/vault
        Wait For Load State    networkidle
        Get Url    contains    /auth/signin
    ```

    - Do NOT use `@playwright/test` — only Robot Framework + Browser library
