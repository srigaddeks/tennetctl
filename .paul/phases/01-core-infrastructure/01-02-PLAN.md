<objective>
## Goal
Running FastAPI backend with all core modules — config, database pool, UUID v7, errors, response envelope, middleware, node registry — and module gating via TENNETCTL_MODULES. GET /health returns the standard envelope.

## Purpose
Every feature vertical (IAM, audit, etc.) imports from 01_core. Without these modules, no feature can define routes, query the DB, generate IDs, or return responses. This is the backbone the entire backend builds on.

## Output
- `backend/01_core/` with 7 core modules
- `backend/main.py` — FastAPI app with lifespan, middleware, health endpoint, module gating
- Updated `requirements.txt` with FastAPI + uvicorn + pydantic + python-dotenv + uuid-utils
- Test suite with 80%+ coverage on 01_core
</objective>

## Key Conventions (from CLAUDE.md)
- `importlib.import_module()` for numeric-prefix dirs — never relative imports
- `conn` not `pool` in service/repo — `pool.acquire()` in routes only
- `uuid-utils` library for UUID v7 — never uuid4()
- Response envelope: `{"ok": true, "data": {...}}` / `{"ok": false, "error": {...}}`
- TENNETCTL_MODULES env var controls module gating (core,iam,audit always on)
- TENNETCTL_SINGLE_TENANT for single-tenant mode
- Start: `cd tennetctl && .venv/bin/python -m uvicorn backend.main:app --port 51734 --host 0.0.0.0 --reload`

    - Frozen dataclass `Config` with fields:
      - database_url: str
      - modules: frozenset[str] (parsed from TENNETCTL_MODULES comma-separated, default "core,iam,audit")
      - single_tenant: bool (from TENNETCTL_SINGLE_TENANT, default False)

    - `load_config() -> Config` — reads from os.environ, loads .env via python-dotenv
    - Config is immutable (frozen=True on dataclass)
    - Do NOT use Pydantic BaseSettings — keep it simple, frozen dataclass

    **backend/01_core/database.py:**
    - `async def create_pool(dsn: str) -> asyncpg.Pool`
    - `async def close_pool(pool: asyncpg.Pool) -> None`
    - These are called from app lifespan in main.py
    - Pool stored on `app.state.pool`
    - Routes call `pool.acquire()` to get conn, pass conn to service/repo

    **backend/01_core/id.py:**
    - `def uuid7() -> str` — returns UUID v7 string via uuid-utils
    - Import: `from uuid_utils import uuid7 as _uuid7`
    - Return `str(_uuid7())`

    **backend/01_core/errors.py:**
    - Base class `AppError(Exception)` with code: str, message: str, status_code: int
    - Subclasses with default status codes:
      - NotFoundError(message, code="NOT_FOUND") → 404
      - ValidationError(message, code="VALIDATION_ERROR") → 422
      - ConflictError(message, code="CONFLICT") → 409
      - ForbiddenError(message, code="FORBIDDEN") → 403
      - UnauthorizedError(message, code="UNAUTHORIZED") → 401

    **backend/01_core/response.py:**
    - `def success(data: Any) -> dict` → {"ok": True, "data": data}
    - `def error(code: str, message: str) -> dict` → {"ok": False, "error": {"code": code, "message": message}}
    - `def paginated(data: list, total: int, limit: int, offset: int) -> dict` → success envelope with pagination meta
    - `def success_response(data: Any) -> JSONResponse` — wraps success() in JSONResponse
    - `def error_response(code: str, message: str, status_code: int) -> JSONResponse` — wraps error() in JSONResponse

    Avoid: ORM, relative imports from numeric dirs, uuid4, mutable config, Pydantic BaseSettings
  </action>
