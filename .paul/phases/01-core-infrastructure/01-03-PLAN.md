  </files>
  <action>
    **Create Next.js project:**
    - Run `npx create-next-app@latest frontend --typescript --tailwind --eslint --app --src-dir --no-import-alias` from project root
    - Or manually scaffold if create-next-app adds too much boilerplate

    **frontend/src/types/api.ts** — ALL shared TS types (one file):
    ```typescript
    type ApiSuccess<T> = { ok: true; data: T }
    type ApiError = { ok: false; error: { code: string; message: string } }
    type ApiResponse<T> = ApiSuccess<T> | ApiError
    type PaginatedResponse<T> = { ok: true; data: T[]; pagination: { total: number; limit: number; offset: number } }
    type HealthData = { status: string }
    ```
    - Export all types
    - No `any` — use `unknown` and narrow
    - String literal unions, not enums

    **frontend/src/lib/api.ts** — typed fetch wrapper:
    - `API_BASE` from env var `NEXT_PUBLIC_API_URL` or default `http://localhost:51734`
    - `async function apiFetch<T>(path: string, options?: RequestInit): Promise<T>`
    - Calls fetch, parses JSON, checks `data.ok`, throws on error
    - Pattern: `if (!data.ok) throw new Error(data.error?.message ?? "Unknown error")`

    **frontend/src/app/layout.tsx:**
    - Root layout with html/body, Inter font from next/font
    - Metadata: title "TennetCTL", description

    **frontend/src/app/page.tsx:**
    - Simple landing page: "TennetCTL" heading, "Developer Platform" subtitle
    - Clean, minimal — Tailwind styled
    - No data fetching yet — just static content

    **frontend/src/app/globals.css:**
    - Tailwind directives (@tailwind base/components/utilities)
    - Minimal custom styles if any

    **frontend/next.config.ts:**
    - API rewrites for dev: `/api/:path*` → `http://localhost:51734/:path*` (avoids CORS in dev)

    **Also add CORS to backend:**
    - Add `fastapi[all]` CORS middleware to backend/main.py
    - Allow origins: ["http://localhost:3000"] in dev
    - This is a small backend change but essential for AC-4

    Avoid: Adding unnecessary pages, over-styling, enum types, any type
