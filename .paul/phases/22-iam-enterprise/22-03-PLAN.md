<objective>
## Goal
SCIM 2.0 compliant `/scim/v2/{org_slug}/Users` + `/Groups` endpoints per RFC 7644. Per-org bearer tokens. Supported ops: list (with filter + pagination), get, create, patch, delete. Idempotency via `externalId`. Deprovisioning (DELETE) deactivates user + revokes sessions.

## Purpose
Okta / Azure AD / Google Workspace / OneLogin push provisioning via SCIM. Without it, enterprises can't auto-onboard/offboard users.
</objective>
