<objective>
## Goal
System-admin-only "login as user" flow with dual-actor audit trail. Red banner in UI during impersonated session. Auto-ends after `impersonation.max_duration_seconds` (policy-driven, default 1800). Every action during the impersonated session emits audit with both `actor_id` (impersonated) and `impersonator_id` (dtl_attr on session).

## Purpose
Support teams need to reproduce user issues without demanding passwords. Auditors require dual-actor trail.
</objective>

<acceptance_criteria>

## AC-1: Impersonation requires system_admin + MFA re-auth
```gherkin
Given caller is system_admin and has TOTP enrolled
When POST /v1/admin/impersonate/{user_id} with body {totp_code}
Then TOTP is re-verified within the last 5 min
And a new session is created for the target user with dtl_attr impersonator_user_id = caller
And audit iam.impersonation.started emitted with {actor, target, reason?}
And the original admin session is preserved (cookie swap on client — two sessions tracked)
```

## AC-2: Audit trail carries both actors
```gherkin
Given an active impersonated session
When any authenticated action occurs
Then audit_scope NodeContext includes both user_id (target) and impersonator_id (admin)
And evt_audit rows during this session have metadata.impersonator_id populated
```

## AC-3: Auto-end after max duration
```gherkin
Given AuthPolicy.resolve(None, "impersonation.max_duration_seconds") = 1800
When 1801 seconds pass since impersonation.started
Then the next request on that session returns 401 with code=IMPERSONATION_EXPIRED
And audit iam.impersonation.expired emitted
And original admin session remains valid
```

## AC-4: Manual end
```gherkin
When POST /v1/admin/end-impersonation is called from the impersonated session
Then that session is revoked
And audit iam.impersonation.ended emitted
And client redirects to admin panel
```

## AC-5: Red banner UI
```gherkin
Given the current session has impersonator_user_id set
Then the topbar shows a persistent red banner "Impersonating {user_email} — [End]"
And every page renders with a red border or visual differentiator
```

```gherkin

When test_impersonation.py + 33_impersonation.robot run
Then MFA enforcement, dual-actor audit, auto-end, manual end all pass
```
