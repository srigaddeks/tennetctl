## Entity IDs
- role: entity_type_id=4 (from dim_entity_types)
- group: entity_type_id=5
- application: entity_type_id=6

## fct_roles specifics
- Has role_type_id SMALLINT FK (system/custom) + nullable org_id
- Roles describe themselves via EAV: code (unique per org), label, description
- v_roles joins dim_role_types for role_type text + pivots EAV attrs

## fct_groups / fct_applications
- Both are org-scoped (org_id NOT NULL)
- Both have EAV: code (unique per org), label, description
- No dim fk beyond org
