<files>backend/02_features/06_notify/sub_features/03_templates/routes.py, backend/02_features/06_notify/sub_features/03_templates/repository.py</files>
<action>
Add `get_template_analytics(conn, *, template_id)` in repo. Runs two queries:
  - delivery status breakdown:
      SELECT status_code, COUNT(*) FROM v_notify_deliveries
      WHERE template_id=$1 GROUP BY status_code
  - event-type breakdown:
      SELECT event_type, COUNT(*) FROM v_notify_delivery_events e
      JOIN v_notify_deliveries d ON d.id=e.delivery_id
      WHERE d.template_id=$1 GROUP BY event_type
Return {by_status: {...}, by_event_type: {...}, total_deliveries: int}.

Add route: GET /v1/notify/templates/{id}/analytics — session-required; returns _response.success({...}).
</action>
