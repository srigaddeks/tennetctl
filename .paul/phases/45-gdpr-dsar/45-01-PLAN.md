# Plan 60-01 — GDPR DSAR Flows

**Phase:** 60 — GDPR DSAR
**Milestone:** v0.8.0 Product Analytics Completion
**Status:** DRAFT
**Type:** Full vertical
**Depends on:** 57 (groups — DSAR cascades to lnk_group_members)

## Scope

Export + delete flows for visitor DSAR requests. Operator-triggered in v1 (no self-service UI yet).

## Tasks

### T1 — Schema (migration 070)
- `evt_dsar_jobs` — job_id UUID v7, job_type (export|delete), visitor_id, actor_user_id, status, row_counts JSONB, started_at, completed_at, error, result_location (for large exports stored to vault/disk)
- `v_dsar_jobs` view

### T2 — New sub-feature `product_ops.dsar`
- 5-file vertical
- Endpoints:
  - `POST /v1/product-visitors/{id}/export` → sync (small) or async (>10k rows)
  - `DELETE /v1/product-visitors/{id}?cascade=true` → hard-delete cascade
  - `GET /v1/dsar-jobs` — job history
  - `GET /v1/dsar-jobs/{id}` — poll status + download result

### T3 — Export service
- Concat-streams JSON across: fct_visitors, dtl_visitor_attrs, lnk_visitor_aliases, evt_product_events, evt_attribution_touches, evt_promo_redemptions, lnk_group_members
- Chunked read by created_at partition to bound memory
- Small: inline response. Large: write to disk + return job_id

### T4 — Delete service
- Single tx: DELETE FROM every product_ops table WHERE visitor_id = X
- Soft-delete fct_visitors (deleted_at); hard-delete all evt_* + dtl_* + lnk_* rows
- Idempotent
- Row counts captured for audit

### T5 — Rate limit
- Reuse existing PG-native rate_limit (Phase 38): 10 DSAR ops per workspace per hour

### T6 — Audit
- `product_ops.dsar.exported` + `product_ops.dsar.deleted` events
- actor_user_id + visitor_id + row_counts_by_table in attributes

### T7 — Admin UI
- `/product/dsar` — job list + "New export" / "New delete" forms + history

### T8 — Live verification (Playwright MCP)
- Seed visitor + 100 events + traits
- Export → JSON contains all rows
- Delete → evt_* rows gone, v_visitors shows deleted_at set, group memberships removed
- Re-run delete → no-op (idempotent)
- Exceed rate limit → 429
