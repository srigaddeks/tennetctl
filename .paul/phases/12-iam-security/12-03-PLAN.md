# Plan 12-03: WebAuthn Passkeys Sub-Feature

**Phase:** 12 (IAM Security Completion)
**Depends on:** 12-01, 12-02
**Autonomous:** true

## Objective

WebAuthn passkeys ‚Äî FIDO2/WebAuthn registration + authentication ceremonies, multiple credentials per user with device naming. Uses `py_webauthn` (webauthn==2.7.1).

## Acceptance Criteria

- [ ] DB: `fct_iam_passkey_challenges` (short-lived, TTL) + `fct_iam_passkey_credentials` (public key, sign_count, aaguid)
- [ ] Backend: register begin/complete + auth begin/complete + list + delete
- [ ] Routes: 6 endpoints under `/v1/auth/passkeys/`
- [ ] pytest: register+auth happy path, invalid challenge, deleted credential
- [ ] Frontend: hooks + `/account/security` TOTP section extended with passkey section

## Tasks

### Task 1: DB migration 031

`"03_iam"."25_fct_iam_passkey_challenges"` ‚Äî (id, user_id nullable, challenge_b64, purpose enum, expires_at, consumed_at, created_at)
`"03_iam"."26_fct_iam_passkey_credentials"` ‚Äî (id, user_id FK, credential_id_b64, public_key_b64, aaguid, sign_count, device_name, last_used_at, created_at, deleted_at)

### Task 2: Backend sub-feature 13_passkeys

`backend/02_features/03_iam/sub_features/13_passkeys/`
- `register_begin`: generate challenge + options, store challenge
- `register_complete`: verify registration, store credential
- `auth_begin`: generate challenge + options (requires email to identify user)  
- `auth_complete`: verify assertion, update sign_count, mint session
- `list_credentials`: list by user_id
- `delete_credential`: soft-delete

### Task 3: Routes + wiring

`POST /v1/auth/passkeys/register/begin`
`POST /v1/auth/passkeys/register/complete`
`POST /v1/auth/passkeys/auth/begin`
`POST /v1/auth/passkeys/auth/complete`
`GET /v1/auth/passkeys`
`DELETE /v1/auth/passkeys/{id}`

### Task 4: Tests

`tests/test_iam_passkeys.py`:
- Register begin ‚Üí gets options with challenge
- Register complete ‚Üí credential stored
- Auth begin ‚Üí gets options
- Auth complete ‚Üí session returned
- Invalid challenge ‚Üí 401
- Deleted credential ‚Üí 404

### Task 5: Frontend

- Hooks: `usePasskeyRegisterBegin`, `usePasskeyRegisterComplete`, `usePasskeyAuthBegin`, `usePasskeyAuthComplete`, `usePasskeyList`, `usePasskeyDelete`
- `/account/security` ‚Äî add passkeys section (enroll + list + delete)
- Signin form ‚Äî add "Passkey" tab
