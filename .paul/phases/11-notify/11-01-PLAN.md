### Task 11 — notify.templates.render node
`backend/02_features/06_notify/sub_features/03_templates/nodes/render_template.py`
- key=`notify.templates.render`, kind=control, tx=caller
- Input: template_key, channel (email/webpush/in_app), variables (dict), org_id
- Fetches template + body from v_notify_templates via repo
- Renders subject + body_html + body_text via Jinja2 Environment(undefined=StrictUndefined)
- Output: rendered_subject, rendered_html, rendered_text (None if no text body)
- Raises if template_key not found or required variable missing
