# Plan 11-04 — Email Channel

## Goal

Send queued email deliveries via aiosmtplib, add pytracking open/click pixels, and handle bounce webhooks.

## Scope

1. `07_email` sub-feature (5 files)
2. Email sender worker (separate asyncio task, started in main.py)
3. Tracking endpoints: `GET /v1/notify/email/track/o/{token}`, `GET /v1/notify/email/track/c/{token}`
4. Bounce webhook: `POST /v1/notify/email/webhooks/bounce`
5. 10+ pytest tests (mocked SMTP)

## Key Design

- **Status flow**: queued(2) → sent(3) / failed(8); open tracking → opened(5); click tracking → clicked(6); bounce webhook → bounced(7)
- **Atomic claiming**: UPDATE SET attempted_at WHERE attempted_at IS NULL ... FOR UPDATE SKIP LOCKED
- **pytracking**: adapt_html() adds tracking pixel + wraps click links; get_open/click_tracking_result decodes tokens
- **Recipient email**: looked up from `03_iam.v_users` by `recipient_user_id`
- **SMTP config**: template → group → smtp_config → vault.get(auth_vault_key)
- **No new migrations needed**
