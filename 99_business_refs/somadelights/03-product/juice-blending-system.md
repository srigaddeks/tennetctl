## 1. Why Modular Blending is Better Than Per-SKU Production

### The Two Approaches Compared

| Dimension | Per-SKU Production | Modular Base System |
|---|---|---|
| **Process** | Buy ingredients for each SKU, juice them together as one batch | Press each ingredient separately into a base juice, store in labeled containers, mix in set ratios per order |
| **Flexibility** | Locked to fixed recipes; new SKU = new recipe from scratch | Unlimited SKUs from the same 8-12 bases; new product = new ratio card |
| **Consistency** | Hard to replicate — depends on exact produce mix each time | Each base is a single ingredient; taste/color/density are predictable |
| **Waste** | Leftover batch of Green Morning has no other use | Leftover spinach base can go into tomorrow's Green Morning, ABC Classic, or a custom profile blend |
| **Customization** | Impossible without separate production per customer | Trivial — same bases, different ratios per wellness profile |
| **Quality Control** | Hard to isolate which ingredient is off in a multi-ingredient blend | Taste each base independently; catch problems before mixing |
| **Speed at Scale** | Linear — 2x orders = 2x juicing sessions | Sub-linear — press bases once, mixing is fast (pour + stir) |
| **Inventory Tracking** | Track per-SKU ingredient lists | Track base volumes; simpler math |

### Recommendation: Modular Base System

The workflow becomes:

```
MORNING BATCH (4:00 - 5:30 AM)
├── Phase 1: PRESS (4:00 - 5:00 AM)
│   ├── Press spinach → Spinach Base jar (labeled, dated)
│   ├── Press cucumber → Cucumber Base jar
│   ├── Press carrot → Carrot Base jar
│   ├── Press beetroot → Beetroot Base jar
│   ├── Press apple → Apple Base jar
│   ├── Press ginger → Ginger Base jar (small batch, concentrated)
│   ├── Press lemon → Lemon Base jar
│   └── Press amla → Amla Base jar (when in season)
│
├── Phase 2: MIX (5:00 - 5:30 AM)
│   ├── Read daily order manifest
│   ├── For each order: measure bases per ratio card
│   ├── Pour into mixing pitcher, stir gently
│   ├── Taste check
│   ├── Pour into delivery bottle, cap, label
│   └── Into cooler bag for delivery
│
└── Phase 3: STORE REMAINING BASES (5:30 AM)
    ├── Seal all base jars
    ├── Refrigerate at 2-4 degrees C
    └── Note remaining volumes for next day planning
```

### Why This Matters for Soma Delights Specifically

1. **Wellness profiles need customization.** A diabetic customer gets zero fruit bases. A kid gets extra apple. Same bases, different ratios. Per-SKU production cannot support this without a separate production line per profile.

2. **Under 2L budget means zero room for waste.** Modular bases let you use every milliliter. Leftover carrot base today becomes tomorrow's ABC Classic or a senior wellness blend.

3. **Education-first brand needs variety.** You want to rotate "Juice of the Week" and seasonal specials. With bases on hand, a new SKU is a new ratio card — not a new recipe, not new ingredients, not a new production workflow.

4. **Quality is everything at launch.** When a customer says "this tastes different from last time," you can trace it to a specific base. Was the spinach base more bitter today? Was the ginger base more concentrated? Isolating bases isolates problems.

## 2. Base Juice Library

### 2.1 Green Bases

#### Spinach Base

| Property | Detail |
|---|---|
| **Ingredient** | Fresh baby spinach (preferred) or mature spinach leaves |
| **Preparation** | Wash thoroughly (3 rinses), remove thick stems, press whole leaves |
| **Yield per kg** | 550-650ml (spinach is ~90% water but compresses heavily in juicer) |
| **Color** | Deep forest green, opaque |
| **Taste profile** | Mild, earthy, slightly metallic (iron), not bitter |
| **Sugar content** | ~0.4g per 100ml (negligible) |
| **Key nutrients** | Iron, folate, vitamin K, vitamin A, calcium, magnesium |
| **Shelf life (refrigerated, sealed)** | 4-6 hours maximum. Spinach oxidizes rapidly — turns brown and bitter. |
| **Oxidation note** | PRESS FIRST in the morning. Use within the same production batch. Do not store overnight. |
| **Best pairs** | Apple (sweetness), lemon (brightness + iron absorption), cucumber (bulk + mild), ginger (warmth) |
| **Storage container** | Dark glass jar or opaque food-grade container to slow light oxidation |
| **Season availability** | Year-round in Hyderabad, best quality in winter (Oct-Feb) |

#### Cucumber Base

| Property | Detail |
|---|---|
| **Ingredient** | English cucumber or regular cucumber |
| **Preparation** | Wash, trim ends, press with skin on (nutrients in skin). No peeling needed. |
| **Yield per kg** | 800-900ml (very high water content, ~96%) |
| **Color** | Pale green, slightly translucent |
| **Taste profile** | Neutral, clean, mildly sweet, refreshing. The "water" of juice blending. |
| **Sugar content** | ~1.7g per 100ml (very low) |
| **Key nutrients** | Potassium, magnesium, vitamin K, silica (skin health) |
| **Shelf life (refrigerated, sealed)** | 8-12 hours. Relatively stable — low oxidation. |
| **Oxidation note** | Slow to oxidize. Can be pressed in middle of session. |
| **Best pairs** | Everything. Cucumber is the universal base — adds volume, hydration, and mild flavor without competing. |
| **Storage container** | Any sealed food-grade container |
| **Season availability** | Year-round. Cheaper in summer. |

#### Celery Base

| Property | Detail |
|---|---|
| **Ingredient** | Fresh celery stalks with leaves |
| **Preparation** | Wash, trim root end, press stalks and leaves together |
| **Yield per kg** | 700-800ml (high water content, ~95%) |
| **Color** | Pale yellow-green, slightly cloudy |
| **Taste profile** | Salty, slightly bitter, herbal. Strong flavor — use as accent, not primary. |
| **Sugar content** | ~1.3g per 100ml (very low) |
| **Key nutrients** | Vitamin K, potassium, folate, antioxidants (apigenin), anti-inflammatory |
| **Shelf life (refrigerated, sealed)** | 6-8 hours. Moderate oxidation. |
| **Oxidation note** | Flavor intensifies (gets more bitter) over time. Use within 6 hours for best taste. |
| **Best pairs** | Apple (balances bitterness), lemon (brightens), cucumber (dilutes intensity), carrot (sweetness) |
| **Storage container** | Sealed glass jar |
| **Season availability** | Year-round in Hyderabad, imported varieties may vary |
| **Max ratio in blend** | 15-20%. Above 20% the salty-bitter flavor dominates. |

### 2.2 Root Bases

#### Carrot Base

| Property | Detail |
|---|---|
| **Ingredient** | Fresh Ooty carrots (preferred — sweeter, deeper color) or Delhi carrots |
| **Preparation** | Wash, scrub (no need to peel if organic), trim tops, press whole |
| **Yield per kg** | 600-700ml (~88% water content) |
| **Color** | Vibrant orange (Ooty) or pale orange (Delhi variety) |
| **Taste profile** | Sweet, earthy, familiar. The most universally liked base. |
| **Sugar content** | ~4.7g per 100ml (moderate — important for diabetic profiles) |
| **Key nutrients** | Beta-carotene (vitamin A precursor), vitamin K, potassium, biotin |
| **Shelf life (refrigerated, sealed)** | 10-12 hours. Root juices are more stable than leafy greens. |
| **Oxidation note** | Minimal oxidation. Color stays vibrant. One of the most stable bases. |
| **Best pairs** | Ginger (classic combo), beetroot (earthy pairing), apple (extra sweetness), orange (citrus-carrot) |
| **Storage container** | Any sealed food-grade container |
| **Season availability** | Year-round. Ooty carrots peak Nov-Feb (better sweetness). |
| **Diabetic profile note** | Include at max 30% in diabetic blends. Has natural sugar but low glycemic index (GI 35-40). Pair with high-fiber vegetables to slow absorption. |

#### Beetroot Base

| Property | Detail |
|---|---|
| **Ingredient** | Fresh red beetroot |
| **Preparation** | Wash, peel (skin can be gritty), trim top and tail, cut into quarters, press |
| **Yield per kg** | 550-650ml (~87% water content) |
| **Color** | Deep crimson-magenta. VERY strong pigment — will stain everything. |
| **Taste profile** | Sweet, earthy, mineral-rich. Distinctive "beet" taste that some love and some find too earthy. |
| **Sugar content** | ~6.8g per 100ml (moderate-high — important for diabetic profiles) |
| **Key nutrients** | Nitrates (blood flow, BP reduction), folate, iron, manganese, vitamin C |
| **Shelf life (refrigerated, sealed)** | 10-12 hours. Very stable base. |
| **Oxidation note** | Minimal. Color actually deepens slightly over time. |
| **Best pairs** | Carrot (classic ABC), apple (sweetness), ginger (warmth), lemon (brightness to counter earthiness) |
| **Storage container** | Dedicated containers — beet stains permanently. Label "BEET ONLY." |
| **Season availability** | Year-round in Hyderabad. |
| **Diabetic profile note** | Despite sugar content, beetroot has a GI of 61 (moderate). Limit to 20-25% in diabetic blends. The nitrate benefits for BP management may outweigh the sugar concern — discuss with customer's doctor. |
| **Performance note** | Beetroot nitrates peak 2-3 hours after consumption. Ideal pre-workout timing. |

#### Ginger Base (Concentrated)

| Property | Detail |
|---|---|
| **Ingredient** | Fresh ginger root |
| **Preparation** | Wash, scrub (no need to peel young ginger; peel older ginger with thick skin), press |
| **Yield per kg** | 250-350ml (very fibrous, low yield) |
| **Color** | Pale yellow, slightly cloudy |
| **Taste profile** | VERY strong, spicy, warming, pungent. This is a CONCENTRATE — never use above 15%. |
| **Sugar content** | ~1.7g per 100ml (low) |
| **Key nutrients** | Gingerol (anti-inflammatory), antioxidants, aids digestion, anti-nausea |
| **Shelf life (refrigerated, sealed)** | 24-48 hours. Ginger is naturally antimicrobial. Most stable base in the library. |
| **Oxidation note** | Very slow. Can be pressed in larger batches and stored for next day. |
| **Best pairs** | Everything — ginger is the universal booster. Especially carrot, apple, lemon, turmeric. |
| **Storage container** | Small glass jar (you won't produce much volume). 250ml jar sufficient for daily batch. |
| **Season availability** | Year-round. |
| **Ratio guidance** | 5% = gentle warmth. 10% = noticeable kick. 15% = strong. Above 15% = too intense for most. Kids: max 5%. Ginger Fire Shot: up to 25% (but it's a 60ml shot, not 250ml drink). |

#### Turmeric Base (Concentrated)

| Property | Detail |
|---|---|
| **Ingredient** | Fresh turmeric root (not powder) |
| **Preparation** | Wash, peel (stains badly — wear gloves), press. Small batches only. |
| **Yield per kg** | 300-400ml (fibrous, moderate yield) |
| **Color** | Deep golden-orange. STAINS EVERYTHING — worse than beetroot. |
| **Taste profile** | Mild, earthy, slightly bitter, warm. Less aggressive than ginger. |
| **Sugar content** | ~0.5g per 100ml (negligible) |
| **Key nutrients** | Curcumin (powerful anti-inflammatory), antioxidants, liver support |
| **Shelf life (refrigerated, sealed)** | 24-36 hours. Relatively stable. |
| **Oxidation note** | Color darkens slightly but taste holds. |
| **Best pairs** | Ginger (classic golden combo), carrot, apple, lemon, black pepper (enhances curcumin absorption by 2000% — add a tiny pinch when mixing, not to the base) |
| **Storage container** | Dedicated "TURMERIC ONLY" container. Will permanently stain. |
| **Season availability** | Year-round in Hyderabad. Fresh turmeric peaks Dec-Mar. |
| **Ratio guidance** | 5% = subtle. 10% = noticeable. Above 10% = bitter for most palates. Standard: 5-10%. |
| **Black pepper note** | Add a pinch of freshly ground black pepper per 250ml serving during mixing (not to the base). This dramatically increases curcumin bioavailability. Mention this to customers as an education point. |

### 2.3 Fruit Bases

#### Apple Base

| Property | Detail |
|---|---|
| **Ingredient** | Fresh apples — Fuji or Royal Gala (sweeter) or Green Shimla (tart, lower sugar) |
| **Preparation** | Wash, remove stickers, cut into quarters (remove core/seeds — apple seeds contain amygdalin), press |
| **Yield per kg** | 650-750ml (~85% water content) |
| **Color** | Pale gold (clear varieties) to light amber. Oxidizes to brown quickly. |
| **Taste profile** | Sweet, clean, universally liked. The "mixer" of the fruit world — makes everything more palatable. |
| **Sugar content** | ~10.4g per 100ml (HIGH — critical for diabetic profiles) |
| **Key nutrients** | Quercetin (antioxidant), vitamin C, potassium, polyphenols |
| **Shelf life (refrigerated, sealed)** | 4-6 hours. Oxidizes fast — turns brown. |
| **Oxidation note** | Add 1 teaspoon lemon juice per 500ml of apple base immediately after pressing. This slows browning significantly (ascorbic acid prevents enzymatic browning). |
| **Best pairs** | Everything — apple is the universal sweetener. Especially spinach (masks bitterness), carrot, beetroot, ginger, lemon. |
| **Storage container** | Sealed container, ideally with minimal headspace (less air = less oxidation) |
| **Season availability** | Year-round. Kashmir/Shimla apples peak Aug-Nov (better quality, lower price). |
| **Diabetic profile note** | EXCLUDE from diabetic profiles or limit to 10% maximum. Apple is the highest-sugar base in the library. Use cucumber or celery as bulk/mixer instead. |
| **Cost note** | Apples are the most expensive base ingredient. Budget Rs 120-180/kg depending on variety and season. |

#### Lemon/Lime Base

| Property | Detail |
|---|---|
| **Ingredient** | Fresh lemons (preferred) or limes (nimbu) |
| **Preparation** | Wash, cut in half, juice by hand or citrus press (do NOT put through cold-press — rind makes it bitter). Strain seeds. |
| **Yield per kg** | 400-500ml (lemons) or 350-450ml (limes) |
| **Color** | Pale yellow (lemon) or pale green (lime), translucent |
| **Taste profile** | Sour, bright, citrusy. The "seasoning" of juice blending — a small amount transforms a bland blend. |
| **Sugar content** | ~2.5g per 100ml (low) |
| **Key nutrients** | Vitamin C (excellent), citric acid (iron absorption), flavonoids |
| **Shelf life (refrigerated, sealed)** | 12-24 hours. Very stable due to acidity. |
| **Oxidation note** | Extremely stable. The most shelf-stable base. Can be pressed for next day. |
| **Best pairs** | Everything. Lemon is the second universal ingredient after ginger. Especially important with iron-rich greens (vitamin C boosts iron absorption 3-6x). |
| **Storage container** | Glass jar. Avoid metal containers (acid reacts). |
| **Season availability** | Year-round in Hyderabad. Nimbu is always available and cheap (Rs 60-80/kg). |
| **Ratio guidance** | 5% = subtle brightness. 10% = noticeably citrusy. Above 10% = too sour for most. Standard: 5-10%. |

#### Orange Base

| Property | Detail |
|---|---|
| **Ingredient** | Fresh Nagpur oranges (sweet) or mosambi (sweet lime) |
| **Preparation** | Peel, separate segments, remove seeds, press. Or cut in half and use citrus press. |
| **Yield per kg** | 500-600ml (after peeling, significant weight is rind) |
| **Color** | Bright orange, slightly cloudy with pulp |
| **Taste profile** | Sweet-tart, familiar, refreshing. More body than lemon, less acidic. |
| **Sugar content** | ~8.4g per 100ml (moderate-high) |
| **Key nutrients** | Vitamin C, folate, potassium, thiamine |
| **Shelf life (refrigerated, sealed)** | 6-8 hours. Moderate oxidation — flavor dulls. |
| **Oxidation note** | Loses brightness after 6 hours. Best pressed same morning. |
| **Best pairs** | Carrot (classic), ginger, turmeric, amla |
| **Storage container** | Sealed glass or food-grade container |
| **Season availability** | Nagpur oranges: Nov-Feb (peak). Mosambi: year-round. |
| **Diabetic profile note** | EXCLUDE from diabetic profiles. High sugar content. |

#### Amla Base (Concentrated)

| Property | Detail |
|---|---|
| **Ingredient** | Fresh Indian gooseberry (amla) |
| **Preparation** | Wash, remove seeds (cut in half, pop out seed), press. Very fibrous — low yield. |
| **Yield per kg** | 350-450ml (fibrous, hard fruit) |
| **Color** | Pale green-yellow, cloudy |
| **Taste profile** | EXTREMELY sour, astringent, slightly bitter. Followed by a sweet aftertaste. Powerful flavor — use sparingly. |
| **Sugar content** | ~4.0g per 100ml (moderate, but the sourness masks it) |
| **Key nutrients** | Vitamin C (one of the richest natural sources — 600mg per 100g), antioxidants, chromium (blood sugar regulation) |
| **Shelf life (refrigerated, sealed)** | 12-24 hours. Very stable due to high vitamin C content acting as preservative. |
| **Oxidation note** | Minimal. Self-preserving due to ascorbic acid content. |
| **Best pairs** | Orange (citrus family), carrot (sweetness balances sour), apple (sweetness), ginger (warmth) |
| **Storage container** | Glass jar. Small quantity — 200-300ml jar sufficient. |
| **Season availability** | Oct-Mar (winter fruit). Not available in summer — can substitute with bottled amla juice (but fresh is far better). |
| **Ratio guidance** | 5% = subtle tang. 10% = noticeable sour. Above 10% = too intense for most. Standard: 5-10%. |
| **Diabetic profile note** | SAFE for diabetic profiles. Despite moderate sugar, amla has been shown to help regulate blood sugar. Include at 5-10%. |

## 4. Ratio Adjustment Rules

### 4.1 Taste Adjustments

| Change | Effect on Taste | Effect on Health | When to Do |
|---|---|---|---|
| +10% apple (replace cucumber) | Sweeter, more palatable, less watery | More sugar (+2.5g), more antioxidants | Customer says "too bland" or "too green" |
| -10% apple, +10% cucumber | Less sweet, more hydrating, lighter | Less sugar (-2.5g), more hydration | Summer blends, diabetic adjustment, weight management |
| +5% ginger (replace cucumber) | Spicier, more warming, more kick | More anti-inflammatory | Winter blends, immunity profile, customer likes spice |
| -5% ginger, +5% apple | Milder, sweeter, less aggressive | Less anti-inflammatory, more sugar | Kids profile, spice-sensitive customers |
| +5% lemon (replace apple) | Brighter, tangier, more acidic | More vitamin C, less sugar | Immunity profile, summer refresh |
| +10% beetroot (replace carrot) | Earthier, deeper color, more mineral | More nitrates, more iron, slightly less sweet | Fitness profile, heart health |
| -all fruit, +vegetable to fill | Earthy, less palatable, very "healthy" | Zero/minimal fruit sugar | Diabetic profile (mandatory adjustment) |
| +5% amla (replace apple) | More sour, more astringent | Massive vitamin C boost | Immunity profile, monsoon season |
| +5% turmeric (replace carrot) | More bitter, more golden | More anti-inflammatory (curcumin) | Joint health, senior profile |

### 4.2 Seasonal Adjustments

| Season | Adjustment | Rationale |
|---|---|---|
| **Summer (Mar-Jun)** | +10% cucumber across all blends, -5% ginger | Hydration focus, less warming ingredients |
| **Monsoon (Jul-Sep)** | +5% ginger, +5% turmeric, +5% amla (where available) | Immunity focus, anti-inflammatory |
| **Winter (Oct-Feb)** | +5% ginger, +5% turmeric, more root bases | Warming ingredients, cold/flu prevention |
| **Amla season (Oct-Mar)** | Add amla to all immunity-focused blends | Fresh amla available — use it while you can |
| **Mango season (Apr-Jun)** | Consider seasonal mango base (future SKU) | Customer demand, local availability, premium pricing |

### 4.3 The Golden Rule of Ratio Adjustment

**Never change more than 2 bases per iteration.** If you change 3+ bases at once and the taste is wrong, you won't know which change caused the problem. Adjust in small increments, taste, adjust again.

## 5. Mixing Protocol (SOP)

### 5.1 Pre-Mixing: Base Pressing Order

Press bases in this EXACT order to manage oxidation:

| Order | Base | Oxidation Speed | Reason for Order |
|---|---|---|---|
| 1 | Spinach | FAST (4-6 hrs) | Press first — it has the shortest usable window |
| 2 | Celery | MODERATE (6-8 hrs) | Leafy greens degrade faster than roots/fruits |
| 3 | Cucumber | SLOW (8-12 hrs) | Stable, but press early to have volume ready |
| 4 | Carrot | SLOW (10-12 hrs) | Very stable root base |
| 5 | Beetroot | SLOW (10-12 hrs) | Very stable. Press AFTER carrots — beet stains the juicer |
| 6 | Apple | FAST (4-6 hrs) | Oxidizes quickly, but adding lemon base helps. Press after roots. |
| 7 | Orange | MODERATE (6-8 hrs) | Citrus press — separate from cold-press juicer |
| 8 | Lemon/Lime | VERY SLOW (12-24 hrs) | Can even be pressed night before |
| 9 | Ginger | VERY SLOW (24-48 hrs) | Can be pressed day before in larger batch |
| 10 | Turmeric | VERY SLOW (24-36 hrs) | Can be pressed day before. CLEAN JUICER AFTER — stains. |
| 11 | Amla | VERY SLOW (12-24 hrs) | Stable. Press last or day before. |

**IMPORTANT:** Clean the juicer screen between beetroot/turmeric and the next base. Beet/turmeric pigment will contaminate lighter-colored bases.

### 5.2 Base Storage After Pressing

1. Pour each pressed juice into its dedicated, labeled container immediately.
2. Label includes: BASE NAME, DATE, TIME PRESSED, VOLUME.
3. Seal with lid — minimize air exposure.
4. Refrigerate at 2-4 degrees C if not using immediately.
5. Add lemon juice to apple base right after pressing (1 tsp per 500ml) to slow browning.

### 5.3 Mixing Steps (Per Bottle)

```
FOR EACH ORDER:

1. READ the order's ratio card (standard SKU card or profile-based mixing sheet).

2. GATHER the required base jars. Check:
   - Is each base within its shelf life? (Check pressed time on label)
   - Does each base look right? (Color check — see QC table in Section 7)
   - Quick sniff test — any off odors?

3. MEASURE each base using measuring cup or digital scale:
   - Measuring cup: for volumes above 50ml (acceptable precision)
   - Digital scale: for volumes under 50ml and for concentrated bases
     (ginger, turmeric, amla) where precision matters
   - 1ml of juice approx 1g — scale and volume are interchangeable

4. POUR bases into mixing pitcher in this order:
   a. Heaviest/thickest base first (beetroot, carrot)
   b. Medium bases next (spinach, cucumber, apple)
   c. Concentrated bases last (ginger, turmeric, amla, lemon)
   Reason: heavy bases at bottom, light ones on top, concentrates last
   so they distribute through the full volume when stirred.

5. ADD any non-juice ingredients:
   - Pinch of black pepper (for blends with turmeric)
   - Pinch of cayenne (for Ginger Fire Shot only)

6. STIR gently with a long spoon or stir rod. 10-15 circular motions.
   DO NOT SHAKE — shaking introduces air, accelerates oxidation,
   creates foam that looks unappetizing.

7. TASTE TEST:
   - Take a small sip (use a clean tasting spoon, not the stir rod)
   - Check: Is the balance right? Too sweet? Too bitter? Too spicy?
   - If off: note what's wrong, adjust by adding 10-15ml of the
     correcting base (see Ratio Adjustment Rules, Section 4)
   - Record any adjustments on the mixing sheet

8. POUR into delivery bottle using funnel (avoid spills and foam).
   Fill to the neck — leave 1cm headspace.

9. CAP the bottle securely.

10. LABEL the bottle:
    - Customer name (if profile-based order)
    - SKU name (e.g., "Green Morning")
    - Date and time of production
    - "Best consumed within 24 hours — keep refrigerated"
    - For profile orders: profile name (e.g., "Diabetic Wellness")

11. PLACE in cooler bag/refrigerator immediately.

12. RECORD on daily production log:
    - Order number
    - SKU/blend name
    - Bases used and volumes
    - Taste check: PASS / ADJUSTED (note what)
    - Time bottled
```

### 7.1 Visual and Taste QC Table

| Base | Fresh Color | Oxidized/Off Color | Max Shelf Hours | Taste When Good | Taste When Off | Action if Off |
|---|---|---|---|---|---|---|
| Spinach | Deep forest green | Brown-green, khaki | 4-6 | Mild, earthy, clean | Bitter, musty, metallic | DISCARD — never serve oxidized spinach |
| Cucumber | Pale green, translucent | Yellow-green, slimy | 8-12 | Clean, neutral, fresh | Sour, fermented | Discard |
| Celery | Pale yellow-green | Dark yellow, brownish | 6-8 | Salty, herbal, clean | Intensely bitter, harsh | Discard |
| Carrot | Vibrant orange | Dull orange, brownish | 10-12 | Sweet, earthy, bright | Flat, stale, slightly sour | Discard |
| Beetroot | Deep crimson | Brownish-crimson (rare) | 10-12 | Sweet, earthy, mineral | Fermented, off-sweet | Discard |
| Apple | Pale gold | Brown, dark amber | 4-6 | Sweet, clean, bright | Flat, mushy taste, vinegary | Discard — browning accelerates fast |
| Orange | Bright orange | Pale, dull orange | 6-8 | Sweet-tart, citrusy | Flat, bitter (pith taste) | Discard |
| Lemon | Pale yellow | Barely changes | 12-24 | Sour, bright, clean | Overly bitter (rare) | Usually fine; discard if bitter |
| Ginger | Pale yellow | Barely changes | 24-48 | Spicy, warming, clean | Fermented (rare) | Usually fine for 48 hrs |
| Turmeric | Deep golden | Darkens slightly | 24-36 | Mild, earthy, warm | Musty, overly bitter | Discard after 36 hrs |

### 7.2 QC Protocol (Daily)

```
BEFORE MIXING SESSION:

1. VISUAL CHECK every base jar:
   - Color within expected range? (See table above)
   - Any separation/sediment? (Normal for some — stir gently. Abnormal if foamy.)
   - Lid sealed properly?

2. SNIFF CHECK every base jar:
   - Fresh, characteristic smell? PROCEED.
   - Sour, fermented, off smell? DISCARD immediately.

3. TIME CHECK:
   - When was this base pressed? (Check label)
   - Is it within shelf life? (See table above)
   - If borderline (within 1 hour of max): taste test before using.

4. TASTE CHECK (if any doubt):
   - Small sip from a clean spoon.
   - Matches expected taste profile? PROCEED.
   - Off in any way? DISCARD. Do not risk customer health.

RULE: When in doubt, throw it out.
Cost of discarding 200ml of base: Rs 15-30.
Cost of serving a bad juice: one lost customer (lifetime value Rs 5,000+).
```

### 7.3 Contamination Prevention

| Risk | Prevention |
|---|---|
| Cross-contamination of flavors | Dedicated containers for strong bases (beet, turmeric). Wash and dry containers between different bases. |
| Bacterial growth | Refrigerate immediately. Never leave bases at room temperature for more than 30 minutes. |
| Oxidation acceleration | Minimize lid-open time. Pour quickly, close immediately. |
| Juicer residue | Clean juicer screen between beetroot/turmeric and lighter bases. Quick rinse with water is sufficient between similar bases. |
| Stale produce | Check produce on arrival. Reject wilted greens, soft apples, rubbery ginger. |

## 9. Scaling the Base System

### 9.1 Scale Stages

| Stage | Daily Orders | Base Pressing | Mixing | Equipment | Staff | Time (Total) |
|---|---|---|---|---|---|---|
| **Launch** (Month 1-2) | 10-20 | Founder presses all bases manually | Founder mixes all orders | Basic juicer + modular kit (Rs 5K) | Founder only | 1.5-2 hrs |
| **Growth** (Month 3-4) | 20-50 | Dedicated press session 4:00-5:00 AM | Mixing session 5:00-5:45 AM | Same equipment, maybe second set of jars | Founder + 1 helper (part-time, Rs 5-8K/mo) | 2-2.5 hrs |
| **Traction** (Month 5-8) | 50-100 | Larger juicer (commercial mini) or batch pressing | Separate mixing station, mixing sheets become critical | Upgrade juicer (Rs 15-25K), more containers | Founder + 1 full-time kitchen staff (Rs 10-12K/mo) | 2.5-3 hrs |
| **Scale** (Month 9-12) | 100-200 | Base production as a dedicated role | Mixing as a dedicated role | Commercial juicer, labeled base dispensers | 2-3 kitchen staff, founder on QC/management | 3-4 hrs |
| **Next Level** (Year 2+) | 200+ | Consider commercial kitchen space | Semi-automated mixing (measured dispensers) | Commercial space, cold storage, possible automation | Team of 4-5 | Move to dedicated facility |

### 9.2 What Changes at Each Stage

**10-20 orders (Launch):**
- Everything is in the founder's head. Mixing sheet is a notebook.
- Quality is controlled by personal taste on every bottle.
- One juicer, one mixing area, one person.
- This is fine. Don't over-engineer at this stage.

**20-50 orders (Growth):**
- Mixing sheets become essential — can't hold 50 recipes in your head.
- Print/write daily mixing sheets the night before.
- A helper can handle washing, cutting, measuring while founder presses and does QC.
- Introduce the daily base volume tracking sheet (Section 6.2).
- Consider pressing ginger and lemon in bulk twice a week (shelf life allows it).

**50-100 orders (Traction):**
- Pressing time becomes the bottleneck. A basic juicer can only do so much per hour.
- Upgrade to a commercial mini cold-press juicer (Rs 15-25K) — 2-3x throughput.
- Kitchen staff can press bases; founder focuses on mixing and QC.
- Introduce formal QC checklists (Section 7.2) — staff needs written standards.
- Base jars need to be bigger — 1L and 2L sizes.

**100-200 orders (Scale):**
- Pressing and mixing are separate roles, separate stations.
- One person presses all bases (4:00-5:30 AM).
- Another person mixes orders from bases (5:30-7:00 AM).
- Founder does QC sampling, manages delivery logistics, handles customer profiles.
- Consider cold storage unit (small commercial fridge) for base staging.

### 9.3 Signs You Need to Level Up

| Signal | What It Means | Action |
|---|---|---|
| Pressing takes >90 minutes | Juicer throughput is maxed | Upgrade juicer or add second unit |
| Mixing session takes >60 minutes | Too many orders for one mixer | Add mixing staff or simplify popular SKUs |
| Taste inconsistency complaints increase | QC is slipping under volume | Formalize QC checklists, add dedicated QC step |
| Bases running out mid-mixing | Under-planning quantities | Use batch planning sheet religiously, increase buffer to 20% |
| Leftover bases consistently wasted | Over-planning quantities | Tighten batch planning, reduce buffer |
| Customer profile orders exceeding standard orders | Personalization demand is real | Invest in mixing sheet system, consider digital tracking |

### 9.4 Never Lose at Scale

No matter how big the operation gets, these principles hold:

1. **Press each ingredient separately.** This is the foundation. Never go back to per-SKU production.
2. **Label everything.** Base name, date, time. No unlabeled jars — ever.
3. **Taste every batch.** Someone with trained palate must taste-check before any bottle ships.
4. **Oxidation-sensitive bases first.** Spinach and apple are always pressed first and used first.
5. **When in doubt, throw it out.** The cost of waste is always less than the cost of a bad customer experience.
