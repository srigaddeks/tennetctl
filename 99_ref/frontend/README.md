## 🛠️ Commands

| Command | Description |
| :--- | :--- |
| `pnpm dev` | Starts all apps and packages in dev mode |
| `pnpm dev:web` | Starts only the Next.js web application |
| `pnpm dev:ui` | Starts the Storybook component library |
| `pnpm build` | Builds all packages and apps |
| `pnpm lint` | Lints the entire workspace |
